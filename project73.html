Print Specifications

Dimensions

  • Final size: 3.5" × 2" (88.9 × 50.8mm)
  • With bleed: 3.75" × 2.25" (95.25 × 57.15mm)
  • Safe area: 3.25" × 1.75" (keep text inside)
  • Resolution: 300 DPI minimum

Color Mode

  • Print: CMYK (C:75 M:65 Y:0 K:0 for brand)
  • Digital: RGB (#6366F1)
  • Pantone: 2726 C (closest match)
  • Black: Rich black C:60 M:40 Y:40 K:100

Typography

  • Font: Inter (all weights)
  • Name: 13-14pt Bold
  • Title: 8-9pt Medium
  • Contact: 7-8pt Regular
  • Tracking: -10 to -25 for headings

File Setup

  • Format: PDF/X-1a:2001
  • Bleed: 0.125" all sides
  • Crop marks: Include
  • Fonts: Outline/Embed all

Usage Guidelines

✓ Do
  • Use high-res 300 DPI files for print
  • Keep text 0.125" from trim edge
  • Use CMYK for offset printing
  • Proof on uncoated stock first
  • Include both front and back designs
✗ Don't
  • Use RGB files for print
  • Place text in bleed area
  • Use images below 300 DPI
  • Stretch or distort the logo
  • Print on glossy without testing
