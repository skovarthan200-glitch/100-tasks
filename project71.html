Logo System

Our logo represents data flowing seamlessly. Use it consistently to build recognition and trust.

Primary Lockup

DF
DataFlow

Logo Mark

DF

Use for app icons, favicons, social avatars, and anywhere space is limited. Minimum 32px.

Dark Background

DF
DataFlow

Always use white logo on dark backgrounds. Never use color logo on dark colors.

Clear Space & Sizing

Clear space: Minimum clear space equals the height of "D" on all sides. This protects logo integrity.
Minimum size digital: 120px wide or 32px tall. Below this, use the mark only.
Minimum size print: 1 inch wide or 0.25 inch tall.

Logo Usage Rules

✓ Do
  • Use approved logo files only
  • Maintain aspect ratio when scaling
  • Ensure sufficient contrast
  • Use full color on light backgrounds
  • Use white on dark backgrounds
✗ Don't
  • Stretch, skew, or rotate
  • Change colors or add gradients
  • Add drop shadows or outlines
  • Place on busy/cluttered backgrounds
  • Recreate or modify the logo

Accessibility

Contrast Requirements (WCAG AA):
  • Normal text: 4.5:1 minimum ratio
  • Large text 18px+: 3:1 minimum ratio
  • UI components: 3:1 against background
  • Primary on white: 8.6:1 ✓ Passes AAA
  • White on primary: 4.5:1 ✓ Passes AA
