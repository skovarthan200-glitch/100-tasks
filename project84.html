Usage Guidelines

✓ Do
  • Use design tokens for all values
  • Maintain 4.5:1 contrast ratio for text
  • Follow 8px spacing grid
  • Use semantic color names in code
  • Test components in dark mode
  • Document component variants in Figma
✗ Don't
  • Hardcode colors or spacing values
  • Mix different button styles in one flow
  • Use more than 2 font weights per screen
  • Create one-off components without tokens
  • Ignore accessibility standards
  • Skip responsive testing
