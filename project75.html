Print Specifications

Dimensions

  • Final size: 8.5" × 11" (portrait)
  • With bleed: 8.75" × 11.25"
  • Safe area: 8" × 10.5"
  • Resolution: 300 DPI minimum
  • File size: 2550 × 3300px at 300 DPI

Color Mode

  • Print: CMYK
  • Primary: C:75 M:65 Y:0 K:0
  • Black: Rich black C:60 M:40 Y:40 K:100
  • Digital: RGB (#6366F1)
  • Pantone: 2726 C (closest match)

Typography

  • Headlines: 36-72pt, Black/ExtraBold
  • Subheads: 18-24pt, Bold/Semibold
  • Body: 10-13pt, Regular/Medium
  • Captions: 8-9pt, Regular
  • Line height: 1.4-1.6 for body

File Setup

  • Format: PDF/X-1a:2001
  • Bleed: 0.125" all sides
  • Crop marks: Include
  • Fonts: Outline or embed all
  • Images: 300 DPI, CMYK

Design Best Practices

✓ Do
  • Keep critical content 0.25" from edges
  • Use high contrast for readability
  • Limit to 2-3 fonts maximum
  • Include clear call-to-action
  • Use hierarchy: size, weight, color
  • Test print on actual stock
✗ Don't
  • Place text in bleed area
  • Use fonts smaller than 7pt
  • Cram too much information
  • Use low-res images under 300 DPI
  • Forget contact information
  • Use more than 3-4 colors
