Print Specifications

Dimensions

  • Flat size: 11" × 8.5" (landscape)
  • Folded: 3.667" × 8.5"
  • With bleed: 11.25" × 8.75"
  • Safe area: 3.42" × 8.25" per panel
  • Resolution: 300 DPI minimum

Color Mode

  • Print: CMYK
  • Primary: C:75 M:65 Y:0 K:0
  • Secondary: C:60 M:70 Y:0 K:0
  • Black: Rich black C:60 M:40 Y:40 K:100
  • Pantone: 2726 C (closest)

Typography

  • Headlines: 22-45pt, Bold/Black
  • Body: 10-13pt, Regular/Medium
  • Captions: 8-9pt, Regular
  • Line height: 1.4-1.6 for body
  • Tracking: -10 to -25 for headlines

File Setup

  • Format: PDF/X-1a:2001
  • Bleed: 0.125" all sides
  • Crop marks: Include
  • Fonts: Outline or embed all
  • Images: 300 DPI, CMYK

Design Best Practices

✓ Do
  • Keep critical content 0.25" from folds
  • Use high contrast for readability
  • Maintain visual hierarchy with size
  • Include clear call-to-action
  • Test print on actual stock first
  • Use vector logos, not raster
✗ Don't
  • Place text across fold lines
  • Use fonts smaller than 7pt
  • Cram too much information
  • Use low-res images under 300 DPI
  • Forget bleed area for backgrounds
  • Use RGB colors for print
