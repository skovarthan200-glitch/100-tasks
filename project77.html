Print Specifications

Dimensions

  • Final size: 3.375" × 2.125" (CR80)
  • With bleed: 3.5" × 2.25"
  • Safe area: 3.125" × 1.875"
  • Resolution: 300 DPI minimum
  • File size: 1012 × 638px at 300 DPI

Color Mode

  • Print: CMYK
  • Primary: C:75 M:65 Y:0 K:0
  • Black: Rich black C:60 M:40 Y:40 K:100
  • Digital: RGB (#6366F1)
  • Pantone: 2726 C (closest)

Typography

  • Name: 16-22pt, ExtraBold
  • Title: 10-12pt, Medium
  • Details: 8-10pt, Regular
  • ID Numbers: 9-11pt, Mono
  • Line height: 1.2-1.4

File Setup

  • Format: PDF/X-1a:2001
  • Bleed: 0.0625" all sides
  • Crop marks: Include
  • Fonts: Outline or embed all
  • Images: 300 DPI, CMYK

Design Best Practices

✓ Do
  • Keep text 0.125" from edges
  • Use high contrast for readability
  • Include photo, name, ID number
  • Add expiration date for security
  • Use QR code for verification
  • Laminate for durability
✗ Don't
  • Use fonts smaller than 7pt
  • Place text in bleed area
  • Use low-res photos under 300 DPI
  • Forget magnetic stripe/chip space
  • Overcrowd with too much info
  • Use more than 2-3 colors
