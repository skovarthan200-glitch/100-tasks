🎯 Entry Points

  • First launch → Splash Screen (01)
  • Returning user → Dashboard (06) if logged in
  • Deep link → Task Detail (07)

🔄 Key Interactions

  • Splash: Auto-advance 2s, fade transition
  • Welcome: Tap "Get Started" → Decision (03)
  • Decision: If new user → Sign Up (04), else Sign In (05)
  • Dashboard: Tap task → Detail (07), Tap FAB → Modal (08)
  • Modal: Submit → Success Toast (09) → Dashboard refresh

⚡ Transitions

  • Screen to Screen: Push Right, 300ms, Ease-Out
  • Modal: Overlay slide up, 300ms, backdrop blur
  • Toast: Slide down from top, 250ms, auto-dismiss 3s
  • Back: Push Left, 300ms

✅ Success States

  • Sign Up complete → Dashboard (06)
  • Task created → Toast (09) → Dashboard refresh
  • Task completed → Checkbox state + strikethrough
  • All fields valid → Submit button enabled

❌ Error States

  • Invalid email → Inline error, red border
  • Network error → Toast: "Connection failed"
  • Empty task name → Disable submit button
  • Session expired → Redirect to Sign In (05)
