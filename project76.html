Print Specifications

Dimensions

  • Final size: 11" × 8.5" landscape
  • With bleed: 11.25" × 8.75"
  • Safe area: 10.5" × 8"
  • Resolution: 300 DPI minimum
  • File size: 3300 × 2550px at 300 DPI

Color Mode

  • Print: CMYK
  • Primary: C:75 M:65 Y:0 K:0
  • Gold accent: C:20 M:30 Y:80 K:0
  • Black: Rich black C:60 M:40 Y:40 K:100
  • Pantone: 2726 C for primary

Typography

  • Titles: 36-60pt, Playfair Display Black
  • Names: 42-60pt, Script or Bold
  • Body: 12-14pt, Inter Regular
  • Captions: 9-11pt, Inter Medium
  • Line height: 1.4-1.8 for body

File Setup

  • Format: PDF/X-1a:2001
  • Bleed: 0.125" all sides
  • Crop marks: Include
  • Fonts: Outline or embed all
  • Images: 300 DPI, CMYK

Design Best Practices

✓ Do
  • Use high-quality 300 DPI images
  • Keep text 0.5" from edges
  • Maintain visual hierarchy
  • Include signature lines
  • Add certificate number/ID
  • Use premium paper stock 80lb+
✗ Don't
  • Use fonts smaller than 9pt
  • Place text in bleed area
  • Use low-res images under 300 DPI
  • Forget date or signatures
  • Overcrowd with too much text
  • Use more than 3 fonts
