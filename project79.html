Print & Digital Specifications

Print Dimensions

  • Standard: 18" × 24" portrait
  • Large: 24" × 36" portrait
  • A3: 11.7" × 16.5" (297×420mm)
  • Bleed: 0.125" all sides
  • Resolution: 300 DPI minimum

Digital Formats

  • LinkedIn: 1200×627px (1.91:1)
  • Facebook Event: 1920×1080px (16:9)
  • Instagram: 1080×1350px (4:5)
  • Email: 600×800px
  • Zoom: 1280×720px (16:9)

Typography

  • Headlines: 72-130pt, Black/ExtraBold
  • Subheads: 24-36pt, Semibold
  • Body: 14-18pt, Regular/Medium
  • Details: 10-14pt, Regular
  • Line height: 0.95-1.5

Color Mode

  • Print: CMYK
  • Primary: C:75 M:65 Y:0 K:0
  • Black: Rich black C:60 M:40 Y:40 K:100
  • Digital: RGB (#6366F1)
  • Pantone: 2726 C (closest)

Webinar Poster Best Practices

✓ Do
  • Lead with clear value proposition
  • Show date/time prominently
  • Include speaker photos & titles
  • Add QR code for easy registration
  • Use action words: "Register", "Join", "Save"
  • Keep text scannable from 10ft away
✗ Don't
  • Bury the date/time in small text
  • Use more than 3 fonts
  • Forget timezone for global audience
  • Overcrowd with too much info
  • Use low-contrast text colors
  • Skip the call-to-action
