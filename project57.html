Parallax Depth

Multiple layers move at different speeds creating 3D depth illusion

Multi-Layer Parallax

Each background layer has a different scroll speed. Back layers move slower, front layers move faster. This creates natural depth perception like looking through a window.

Key technique: Use data-speed attribute to control parallax intensity. Values 0.1-0.9 work best. Higher values = faster movement.

Mouse Parallax

Elements follow your cursor with different depths. Move your mouse to see the effect.

Interactive Parallax

Mouse movement drives parallax instead of scroll. Each element has a data-depth value determining how much it moves. Creates engaging, responsive backgrounds.

Implementation Notes

Performance: Use transform and opacity only. Never animate width, height, or top/left. Add will-change: transform to animated elements.

Smooth Scrolling: Use scroll-behavior: smooth on html for anchor links. For JS-driven scroll, use requestAnimationFrame.

Mobile: Parallax can be janky on mobile. Consider disabling complex effects below 768px or reducing layer count.

Accessibility: Respect prefers-reduced-motion. Provide static fallbacks for users who disable animations.
