Print Specifications

Dimensions

  • Standard: 18" × 24" (portrait)
  • Large: 24" × 36" (portrait)
  • A3: 11.7" × 16.5" (297×420mm)
  • Bleed: 0.125" all sides
  • Resolution: 300 DPI minimum

Color Mode

  • Print: CMYK
  • Primary: C:75 M:65 Y:0 K:0
  • Black: Rich black C:60 M:40 Y:40 K:100
  • Digital: RGB (#6366F1)
  • Pantone: 2726 C (closest)

Typography

  • Headlines: 72-130pt, Black/ExtraBold
  • Subheads: 24-36pt, Bold/Semibold
  • Body: 14-18pt, Regular/Medium
  • Captions: 10-12pt, Regular
  • Line height: 1.1-1.5

File Setup

  • Format: PDF/X-1a:2001
  • Bleed: 0.125" all sides
  • Crop marks: Include
  • Fonts: Outline or embed all
  • Images: 300 DPI, CMYK

Design Best Practices

✓ Do
  • Use hierarchy: 1 main message
  • Keep text 0.5" from edges
  • High contrast for readability
  • Include date, time, location
  • Clear call-to-action
  • Test print at 100% scale
✗ Don't
  • Use fonts smaller than 12pt
  • Place text in bleed area
  • Use low-res images under 300 DPI
  • Forget contact information
  • Overcrowd with too much text
  • Use more than 3 font families
