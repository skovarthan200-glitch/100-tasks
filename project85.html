Component States

Button States

Define all interactive states in XD: Default, Hover, Active, Disabled, Loading.

Default
Base state. BG: #6366F1
Hover
Cursor over. BG: #4F46E5
Active
Pressed. Scale: 0.98
Disabled
Opacity: 50%. Cursor: not-allowed
XD Setup: Component States 1. Create Master Component: Button/Primary 2. Add State: Hover (change background) 3. Add State: Active (add scale transform) 4. Add State: Disabled (reduce opacity) 5. Set Trigger: Hover, Tap 6. Transition: Dissolve, 200ms, Ease-Out

Adobe XD Best Practices

✓ Do
  • Use Components for all repeatable elements
  • Create Component States for interactions
  • Use Repeat Grid for lists and galleries
  • Set up Auto-Animate for micro-interactions
  • Use Stacks for responsive layouts
  • Name layers semantically: Button/Primary
  • Use Character Styles for typography
  • Share CC Libraries for team consistency
✗ Don't
  • Detach components unnecessarily
  • Use absolute positioning for layouts
  • Hardcode values instead of using styles
  • Mix artboard sizes in one file
  • Forget to set prototype interactions
  • Skip responsive resize testing
  • Use too many artboards per file
  • Ignore 8px grid system
